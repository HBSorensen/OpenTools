# Test Board
This board is a proof of concept for making the Flux project with the following features:

1. Ethernet connection on the back of the PCB for network connection
2. USB 2.0 (A connector) host on the back of the PCB for connecting to a USB printer (Vevor Y468BT)
3. USB 2.0 (A connector) host on the front of the PCB for connecting to an external USB connector
4. I2C (IDC style) connector on the PCB for connecting to Card reader and Display
5. SPI (IDC style) connectors on the PCB for connecting 2 SPI clients.
6. Raspberry Pi 5 style 40 pin connector (if possible) for allowing standard Raspberry Pi hats to be connected.
7. Powered by external 12V power supply and internal power regulators where needed.
8. Status LEDs on board for each Power levels (3.3V, 5V and 12V as a minimum). 
9. 3 RGB LEDS controlled by the VAR-SOM board.
10. Micro SD card reader on the front of the PCB
11. Board dimensions (and front/back connector placements) should be in x.0 mm with M3 mounting holes 4 mm (both x and y) from the edge of the PCB.
